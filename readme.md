![build](https://api.travis-ci.org/JichouP/chat_app.svg?branch=master)

## なにこれ

+ チャットアプリケーション
  + LINE的なのを作る

## インストール

1. Clone this repository
2. `yarn install`
3. `yarn start` でクライアントウォッチ開始
4. `yarn node` でサーバーウォッチ開始 `localhost:3000` にサーバーを立てる
